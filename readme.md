## Microserviços em GO

Estudo realizado para aprendizagem, criando micro serviços para no futuro criar um aplicação com docker, banco de dados, e kafka para eventos. - [front-end](https://github.com/Ghenoo/front-microgo) sendo criado.

## Ferramentas e Pacotes

- [Gin-gonic](https://github.com/gin-gonic/gin) -> Web Server
- [Gorm](https://gorm.io/index.html) -> ORM
- [Air](https://github.com/air-verse/air) -> refill console
